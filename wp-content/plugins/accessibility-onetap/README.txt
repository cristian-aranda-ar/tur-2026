== Installation ==

1. Open WordPress Dashboard:
Log in to your WordPress dashboard.

2. Access Plugin Menu:
On the left panel, click "Plugins" and select "Add New."

3. Search for "Accessibility Widget by OneTap вАУ Easy One-Click Accessibility Toolbar":
On the Add Plugin page, locate the search box in the top right corner.
Type "Accessibility Widget by OneTap вАУ Easy One-Click Accessibility Toolbar" in the search box.

4. Install the Plugin:
After finding the "Accessibility Widget by OneTap вАУ Easy One-Click Accessibility Toolbar" plugin, click the "Install Now" button below it.

5. Activate the Plugin:
Once the installation is complete, click the "Activate" button that appears.

6. You're Ready to Use the Plugin:
After activation, navigate to the "Accessibility Widget by OneTap вАУ Easy One-Click Accessibility Toolbar" section in your WordPress dashboard to start importing and exporting menus.
